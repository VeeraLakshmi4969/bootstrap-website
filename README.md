# 🚀 Bootstrap Project

A responsive and modern website built using **HTML5**, **CSS3**, and **Bootstrap 5**. This project demonstrates the use of Bootstrap's grid system, responsive utilities, and various UI components to create a clean and mobile-friendly web page.

## 📌 Features

- ✅ Responsive Navigation Bar
- ✅ Bootstrap Carousel
- ✅ Clients Section
- ✅ Progress Bars
- ✅ Contact Form
- ✅ Embedded Google Maps
- ✅ FAQ Accordion
- ✅ Responsive Footer
- ✅ Mobile-Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

1. Clone the repository

```bash
git clone https://github.com/VeeraLakshmi4969/Bootstrap-project.git
```

2. Open the project folder.
